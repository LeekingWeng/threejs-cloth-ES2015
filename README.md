# threejs-cloth-ES6

[Cloth example](http://threejs.org/examples/#webgl_animation_cloth) from **threejs** rewritten with **ES6** features.

Using the [threejs ES6 starter](https://github.com/nikpundik/threejs-starter-npm) 

Instructions:
```
npm install
npm run build
npm start
```
